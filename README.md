# Plannily - AI-First Travel Planner

Plannily is an AI-powered travel platform that helps you plan your perfect trip.

## Features

- **Discovery Flow**: "I don't know where to go" - Get personalized recommendations based on your vibe.
- **Planner Flow**: "I know where to go" - Generate optimized day-by-day itineraries.
- **AI Itinerary**: Smart scheduling, activity suggestions, and real-time adjustments.

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Start the development server:
    ```bash
    npm run dev
    ```

3.  Open [http://localhost:5173](http://localhost:5173) in your browser.
